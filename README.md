# Proyecto Final de Programación Web 1

Este repositorio contiene el proyecto integrador de **Programacion Web 1**, desarrollado de manera individual porque no tenia con quien hacerlo porque lo empece a hacer muy tarde.
El objetivo es poner en practica la teoria dada en clase y los lenguajes aprendidos para crear un **Sitio Web**.

---

## Descripción

El proyecto consiste en el **desarrollo de un sitio web** que integra:

- Estructura en **HTML5**
- Estilos con **CSS**
- Interactividad con **JavaScript**
- **Buenas prácticas de diseño y usabilidad**
